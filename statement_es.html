Declaración de servicio

Usted comprende y acepta que si utiliza el servicio de detección de máscara facial, se considera que ha leído completamente y acepta aceptar el contenido de esta declaración; de lo contrario, detenga este servicio de inmediato.

1. Contenido del servicio

Este servicio se proporciona a los usuarios en forma de página web. Puede obtener el servicio abriendo la página web para realizar la función de detectar y recordar automáticamente el uso de máscaras.

2. Protección de la privacidad

2.1 Recolección de datos

Usted comprende y acepta que, para brindar servicios, acepta encender la cámara en la página web e identificar el contenido capturado. La página web no recopila información sobre el uso de máscaras.

2.2 Almacenamiento y protección de datos

Este servicio no almacena ni carga el contenido de video, audio e imagen capturado. El proceso de detección se realiza en tiempo real en su navegador de forma local.

3. Declaración y Garantía

Debe asegurarse de que su uso de este servicio cumpla con los requisitos de las leyes y regulaciones, y usted es responsable de la legalidad y el cumplimiento de sus fuentes de contenido y adquisiciones. Este servicio solo le brinda un servicio de detección de máscara facial de acuerdo con sus instrucciones y no es responsable del cumplimiento legal de su uso de este servicio.

4. Notificación y corrección

Usted comprende y acepta que, para satisfacer las necesidades de las políticas y los negocios, actualizaremos el contenido de esta declaración de servicio cuando sea necesario y la publicaremos en este sitio web. Consulte esta declaración de servicio para obtener actualizaciones a tiempo.
